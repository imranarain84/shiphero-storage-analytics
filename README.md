# ShipHero Storage Analytics Engine

A warehouse storage cost reporting tool built with Python and Streamlit.

## Components
- **Nightly Pull** — Runs at 2am, pulls ShipHero inventory, saves to DigitalOcean Spaces
- **Streamlit App** — Reporting UI that reads from archived snapshots
- **Admin Panel** — Password-protected tag manager at /Admin
